---

## 🖼️ Imágenes e iconos

Colocá tus imágenes en la carpeta `images/` con estos nombres exactos:
- `imagen_1.jpg` — Hero principal (cockpit / simulador desde atrás)
- `imagen_2.jpg` — Sección comunidad (pilotos compitiendo)
- `imagen_3.jpg` — Sección simuladores (equipo completo)

Colocá tus íconos SVG en la carpeta `icons/` con los nombres indicados arriba.

> **Tip:** Si no tenés íconos SVG, podés usar iconos gratuitos de [heroicons.com](https://heroicons.com) o [tabler-icons.io](https://tabler-icons.io) — son perfectos para este estilo.

---

## 🚀 Publicar en GitHub Pages

1. Creá un repositorio nuevo en GitHub (ej: `apex-simracing`)
2. Subí todos los archivos de esta carpeta
3. Andá a **Settings → Pages**
4. En **Source** seleccioná `main` branch y carpeta `/ (root)`
5. Guardá — en minutos tu sitio estará en: `https://tuusuario.github.io/apex-simracing`

- **Responsive** — Mobile, tablet y desktop
- **Animaciones** — Scroll reveal, hover effects, navbar sticky
